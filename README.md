## 部署步骤

1. 环境准备：安装Python 3.8及以上版本，Mysql数据库。
2. 克隆代码到本地：`git clone [仓库地址]`
3. 安装依赖：`pip install -r requirements.txt`
4. 数据库迁移：`python manage.py makemigrations && python manage.py migrate`
5. 运行服务：`python manage.py runserver`
